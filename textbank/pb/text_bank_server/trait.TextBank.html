Inserts or updates a text translation with optional ID specification.

Supports two modes of operation:

  • Insert mode: When text_id is 0/unset, performs idempotent insert by content. Returns existing ID if (language, text) pair already exists, otherwise allocates new ID.
  • Update mode: When text_id is specified, updates or inserts text for that specific ID.

All text content is automatically normalized using Unicode NFC normalization to ensure consistent representation regardless of input encoding.

Retrieves a text translation by ID with optional language filtering.

When language is specified, returns the text in that specific language. When language is empty/unset, returns text in a preferred language (English first, then any available language).

Searches text content using regular expressions with optional language filtering.

Supports both language-specific and cross-language search:

  • When language is specified, searches only within that language
  • When language is empty, searches across all languages (preferring English results)

The regex pattern is applied to normalized (Unicode NFC) text content. Uses Rust regex syntax for pattern matching.
